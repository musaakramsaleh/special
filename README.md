<h1>Special Project</h1>
<h3>Here we mainly focused on making a ui where user can get a lot of suggestions for searching like in a booking or a travel site</h3>
<h3>Key Features:</h3>
<li>Firebase has been used to ensure user login and registration for both google and github</li>
<li>category and brandname filtering</li>
<li>Sort functionality with date and price</li><br>
<p>Live site: <a href="https://my-art-72b96.web.app">https://my-art-72b96.web.app</a></p>
